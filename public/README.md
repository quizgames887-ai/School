# Public Assets

This directory contains static assets that are served by Next.js.

## Logo

Please add the Alahed International Schools logo file here as `logo.png`.

The logo should be:
- A circular emblem with blue and white colors
- High resolution (at least 200x200 pixels, preferably 400x400 or higher)
- PNG format with transparent background (if possible)
- Named exactly `logo.png`

The logo will be displayed in:
- Navigation bar (top left)
- Login page
- Signup page
- Other pages throughout the application
